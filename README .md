## Overview
Brainex is a web-based quiz game built with Python and Flask.  
It challenges users with questions from 12 different topics (India, Asia, International, Cricket, Football, Kabaddi, Math, Science, SST, GK, Technology, Mahabharata/Ramayana).  

- Users advance through questions by choosing the correct answer.  
- If the user selects a wrong answer, the game ends immediately.  
- If the user answers all questions correctly, they are rewarded with a trophy!  

---

## Features
- Dynamic questions loaded from Python (`questions.py`)  
- Correct / Wrong answer logic  
- Score tracking  
- Result page shows score or trophy  
- Responsive design for mobile & desktop  
- Smooth animations for buttons and question transitions
